# ✈️ Mini Flight Reservation System (Java)

A simple Flight Reservation System developed in Java using one-dimensional arrays and methods.

This project was created to improve my Java programming skills before learning Object-Oriented Programming (OOP). The application runs in the console and simulates a basic flight reservation process.

---

## Features

- Login System
- Show Flight List
- Search Destination
- Reserve Flight
- Cancel Reservation
- Passenger Check-in
- Show Checked-in Passengers
- Show Pending Passengers
- Count Checked-in Passengers
- Count Pending Passengers
- Calculate Occupancy Rate
- Show Most Expensive Flight
- Show Cheapest Flight
- Calculate Average Ticket Price
- Calculate Total Income
- Show Flights Above Average Price
- Show Longest Destination Name
- Show Shortest Passenger Name
- Search City
- Show Domestic Flights
- Show International Flights
- Show Flights Starting With A
- Show Flights Ending With N
- Show Longest Flight Number
- Show Top 3 Most Expensive Tickets
- Flight Summary
